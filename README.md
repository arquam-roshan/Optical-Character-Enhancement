# Optical-Character-Enhancement

Python project to enhance character reading by highlighting textual based documents for easier recognition of characters by OCR technologies

To get started follow the steps listed below:

1. Clone the project on to your local machine
2. Ensure python environment is setup
3. Create a virtual environment by running the following command:
   python -m venv venv
4. source venv/bin/activate
5. Install the requirements.txt file by running the following command:
   pip install -r requirements.txt
6. Configure VS Code to Use the Virtual Environment:
   Press Cmd+Shift+P to open the Command Palette.
   Type "Python: Select Interpreter" and select the Python interpreter from your virtual environment (venv).
7. Run the python script:
   Python image_processing.py
